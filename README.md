# Quel Vêtement - Site (page d'accueil)

Site statique de la page d'accueil de **quel-vetement.com**.

- **Français**, **autonome** (polices, CSS, JS et images en local), **0 requête externe**.
- Sans tracking / pub / consentement.
- **Sans trace WordPress** : aucun fingerprint (generator, feeds, REST, emoji, Yoast…),
  dossiers `assets/` et `includes/` (plus de `wp-content` / `wp-includes`).

## Déploiement

Chemins d'assets en **racine** (`/assets/...`, `/includes/...`, `/__ext/...`). Servir à la
**racine d'un domaine** (ou domaine personnalisé). Depuis un sous-chemin, les chemins absolus
ne résolvent pas.

Aperçu en ligne : https://quel-vetement-home.pages.dev/
